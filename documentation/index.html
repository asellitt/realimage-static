Atrium - Responsive Corporate One Page Template

　　　 　　　　　　　　　　　

Atrium is a parallax, responsive corporate one page template, suitable for business or corporate related projects.
The Template is maintained in a minimalist and modern style with strong green accents. It has a responsive layout that looks great on mobile and tablet devices.

The Template comes with hero unit based on responsive fullscreen slideshow that perfectly adapts to any display size both horizontally and vertically.
Next is descriptive about section with small slider and carousel of key features with parallax background.
Services is the next section with features list and 86 different icons to choose from. Just below is percentage counters unit with parallax background.
The next section is filterable portfolio with support for youtube, vimeo, metacafe and dailymotion videos.
Just below is client's testimonials section in a form of carousel.
The team section comes with animated skills progress bars.
There is also blog unit with a separate blog page with sidebar and a single post page with comments.
The next sections are the latest tweets carousel and responsive pricing plans with some additional features and FAQ accordion.
The contact section comes with map of location and working contact form with 34 social icons to choose from.

The Template comes with PSD Source Files.

Project structure

This template comes in two independent versions: HTML and PHP.
Due the fact that HTML version doesn't include PHP code, some features are available only in PHP version.

Plugins

Contact form

This project includes a contact form ready for use.
All you need is to configure some constants located in file config.php:

Required data

Constant Description
CONTACT_FORM_TO_NAME name of recipient eg. "My Company"
CONTACT_FORM_TO_EMAIL e-mail address of recipent eg. "mycompany@gmail.com"

Optional SMTP settings

Constant Description
CONTACT_FORM_SMTP_PORT your SMTP port eg. "587"
CONTACT_FORM_SMTP_HOST your SMTP host eg. "smtp.gmail.com"
CONTACT_FORM_SMTP_USER login to your e-mail account eg. "mycompany"
CONTACT_FORM_SMTP_PASSWORD password to your e-mail account
CONTACT_FORM_SMTP_SECURE secure connection protocol: "SSL" or "TLS"

Texts and messages

Constant Description
CONTACT_FORM_SUBJECT subject of e-mail message
CONTACT_FORM_MSG_INVALID_DATA_NAME text displayed when name is empty
CONTACT_FORM_MSG_INVALID_DATA_MAIL text displayed when e-mail address is invalid
CONTACT_FORM_MSG_INVALID_DATA_MESSAGE text displayed when message field is empty
CONTACT_FORM_SEND_MSG_OK text displayed when e-mail was sent successfully
CONTACT_FORM_SEND_MSG_ERROR text displayed when e-mail cannot be sent


Twitter user timeline

This project includes a twitter user timeline ready for use.
All you need is to configure some constants located in file config.php:

Required data

Constant Description
TWITTER_USER_TIMELINE_USER the screen name of the user for whom to return latest tweets
TWITTER_USER_TIMELINE_COUNT specifies the number of tweets to try and retrieve, up to a maximum of 200
TWITTER_USER_TIMELINE_CONSUMER_KEY consumer key
TWITTER_USER_TIMELINE_CONSUMER_SECRET consumer secret key
TWITTER_USER_TIMELINE_ACCESS_TOKEN access token
TWITTER_USER_TIMELINE_ACCESS_TOKEN_SECRET access secret token

FAQ

Installing template

To install this template upload all files from www/php or www/html folder into HTTP server.
Due the fact that HTML version doesn't include PHP code, some features (Contact form, Twitter user timeline) are available only in PHP version.
It is highly recommended to use PHP version instead of HTML (if it is possible).


Setup Google Maps

To configure coordinates of Google Maps, follow these steps:
1. Open config.php and setup coordinates: 

define('GOOGLE_MAPS_COORDINATES_LAT','53.276698');
define('GOOGLE_MAPS_COORDINATES_LNG','-6.12562');


Setup contact form

To configure contact form, follow these steps:
1. Open config.php and setup all necessary data (more informations you can find in tab Plugins / Contact Form).

Setup Twitter user timeline

To configure Twitter user timeline, follow these steps:
1. Create and setup your Twitter Account Application. You can learn more about it here.
2. Open config.php and setup all necessary data (more informations you can find in tab Plugins / Twitter User Timeline).


Enable/disable animations

To disable animations (during scrolling) on tablet devices you have to set constant WAYPOINT_ANIMATION_TABLET_ENABLE in file config.php to 0:

define('WAYPOINT_ANIMATION_TABLET_ENABLE',0);

To disable animations (during scrolling) on mobile devices you have to set constant WAYPOINT_ANIMATION_MOBILE_ENABLE in file config.php to 0:

define('WAYPOINT_ANIMATION_MOBILE_ENABLE',0);

To disable animations (during scrolling) on other devices (desktop) you have to set constant WAYPOINT_ANIMATION_ENABLE in file config.php to 0:

define('WAYPOINT_ANIMATION_ENABLE',0);


Enable/disable parallax effect

To disable parallax effect on tablet devices you have to set constant PARALLAX_TABLET_ENABLE in file config.php to 0:

define('PARALLAX_TABLET_ENABLE',0);

To disable parallax effect on mobile devices you have to set constant PARALLAX_MOBILE_ENABLE in file config.php to 0:

define('PARALLAX_MOBILE_ENABLE',0);

To disable parallax effect on other devices (desktop) you have to set constant PARALLAX_ENABLE in file config.php to 0:

define('PARALLAX_ENABLE',0);
